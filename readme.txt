Это игра, где главный смысл состоит в том, что бы доска снизу экрана отбивала мяч в блоки/другие объекты.
Управлять доской можно с помощью клавиш - стрелок на клавиатуре.
Также есть дополнительное ускорение на клавишу SHIFT.
В одиночном режиме есть 3 вида сложности:
-"Легчайшая" - стандартная сложность, без излишеств
-"Сложная" - режим без ускорения на клавишу SHIFT
-"Очень сложная" - режим с ускоренным шаром
Также есть второй режим игры - против ИИ, в таком случае победа достигается тогда, когда ИИ пропускает мяч за верхнюю границу поля, а поражение тогда, когда мяч уходит
за нижнюю.
